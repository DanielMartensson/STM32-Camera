# STM32 Ethernet camera

This project have:

 * CAN-bus
 * Encoder
 * Input capture
 * DAC 12-bit
 * ADC 12-bit
 * Digital N-channel output
 * Digital input
 * FSCM 16-Bit LCD
 * DCMI camera
 * Ethernet

This project will focus on

 * OV7670 Camera SCCB protocol for computer vision
 * Ethernet for STM32F4 series
